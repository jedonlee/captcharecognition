# 参数与数据校验

## 函数入口校验
- 公开函数检查参数类型和范围，概率值限定 [0, 1]
- 示例：`assert 0 <= p <= 1, f"概率值 {p} 不在 [0,1]"`
- 使用 TypeError 而非 AssertionError 处理类型错误

## CTC 专属校验
- 计算 CTC Loss 前必须检查：
  - `input_lengths` 所有值 ≥ 对应 `target_lengths`
  - `input_lengths` 最大值 ≤ 模型输出的时间步数（当前为 16）
  - 标签索引不超出 [0, 62]，空白符固定为 62

## 张量一致性
- 模型输入必须 (B, 3, 64, 256)，float32，值域 [0, 1]
- 模型输出 encoder_out 必须 (T, B, 63) 并已 log_softmax
- 操作前后确认设备一致（训练在 cuda，验证时可切换）

## 数据质量检查
- 图像加载后检查是否为 None
- 图像尺寸是否符合预期
- 标签文件是否存在且非空
- 标签长度是否在 [1, max_length] 范围内

## 生成器质量检查
- 每张生成的图必须通过 Laplacian 方差 ≥ 30 且对比度 ≥ 15
- 失败自动重试，最多 5 次，仍失败则警告并跳过
- 生成统计信息必须记录（成功数、失败数、平均质量分数）

## 配置参数校验
- 训练启动前校验所有超参：
  - learning_rate > 0
  - batch_size > 0
  - num_epochs > 0
  - 0 < ctc_weight + ce_weight <= 1
  - 0 <= label_smoothing < 1
- 校验失败立即报错，不能静默使用默认值
